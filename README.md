# アニメキャラあてゲーム

複数人でリアルタイムに楽しむアニメキャラ当てクイズゲームです。出題者と回答者に役割が分かれ、回答は口頭で行います。

## 技術スタック

| レイヤー | 技術 |
|---|---|
| フロントエンド | React + TypeScript + Tailwind CSS |
| リアルタイム通信 | Socket.io (WebSocket) |
| バックエンド | Node.js + Express |

## ゲームの流れ

1. 出題者がルームを作成 → 4桁のルームコードが発行される
2. 回答者がルームコードを入力して参加
3. 出題者がゲームを開始（問題数・制限時間を設定）
4. キャラクター画像が全員に表示される
5. 回答者が口頭で答え、出題者が正解/不正解を判定
6. 全問終了後、スコアランキングを表示

## セットアップ

```bash
# 依存関係をすべてインストール
npm run install:all

# 開発サーバーを起動（フロント + バックエンド同時起動）
npm run dev
```

- フロントエンド: http://localhost:5173
- バックエンド: http://localhost:3000

## ディレクトリ構成

```
anime-quiz/
├── server/          # Express + Socket.io サーバー
├── client/          # React フロントエンド
├── scripts/         # データ生成スクリプト
└── package.json     # ルート（concurrently で同時起動）
```
